# Overview
This program is a Simple Expense Tracker written in Rust. It allows users to manage their expenses by adding expense details, viewing all recorded expenses, and calculating total expenses. The application features a menu-driven interface where users interact with the program via numeric choices.

## Features
Add an Expense: Enter details such as description, quantity, price, and date of an expense.
Show All Expenses: Display all recorded expenses with their details.
Calculate Total Expenses: Calculate the total of all expenses.
Exit: Quit the application
